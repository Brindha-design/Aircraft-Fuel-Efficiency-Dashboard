# ✈️ Aircraft Fuel Efficiency Dashboard

## 📌 Overview
This project is an interactive Power BI dashboard that analyzes aircraft fuel efficiency and operational performance.

## 🛠️ Tools Used
* **Power BI**
* **Excel**
* **DAX**

## 📊 Dashboard Features
* **Fuel Consumption Analysis**
* **Distance Travelled Analysis**
* **Aircraft-wise Comparison** (A320 vs B737)
* **Interactive Filters & Slicers**
* **KPI Cards**

## 💼 Business Value
This dashboard helps monitor aircraft fuel usage, compare efficiency between aircraft types, and support operational decision-making.
